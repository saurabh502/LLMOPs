Introduction to Python

Python is a high-level, interpreted, general-purpose programming language known for its readability, simplicity, and extensive libraries. It's widely used in web development, data science, machine learning, scripting, automation, and much more.

Prerequisites:

  • Python Installation: Ensure you have Python 3 installed from python.org
  • Code Editor or IDE: VS Code, PyCharm, Sublime Text, or online platforms like Google Colab

Your First Python Code

The traditional first program is "Hello, World!":

# This is a comment. Python ignores lines starting with #
print("Hello, World!")

Save this as hello.py and run it with:

python hello.py

The print() function is a built-in Python function used to display output to the console.

Python Fundamentals

1.1 Variables

Variables are names given to memory locations used to store data values. Think of them as labeled boxes where you can put information.

Key Points:

  • Created with = operator
  • Dynamic typing (no type declaration needed)
  • Case-sensitive names
  • Cannot use Python keywords

Naming Rules:

  • Start with letter or underscore
  • Can contain letters, numbers, underscores
  • No spaces or special characters
  • Descriptive names recommended
# Variable assignment examples
message = "Learning Python is fun!" # A string variable
count = 15 # An integer variable
price = 19.99 # A float variable
is_valid = True # A boolean variable

# Variables can be reassigned to different values (and types)
count = 20
print("New count:", count)
count = "Twenty" # Now count holds a string
print("Count is now:", count)

1.2 Data Types

Data types define the kind of value a variable can hold and the operations that can be performed on it.

Type Description Example
int Integer (whole numbers) 10, -5, 0
float Floating-point number 3.14, -0.01
str String (text) "Hello", 'Python'
bool Boolean (True/False) True, False
list Mutable sequence [1, "apple", 3.5]
tuple Immutable sequence (1, "apple", 3.5)
dict Key-value pairs {"name": "Bob", "age": 25}
set Unordered unique elements {1, 2, 3}
# Checking data types with type()
num = 10
pi = 3.14159
greeting = "Good morning"
my_list = [10, 20, 30]
my_tuple = (10, 20, 30)
my_dict = {'a': 1, 'b': 2}
my_set = {1, 2, 2, 3} # Duplicates are ignored

print(type(num)) # Output: <class 'int'>
print(type(pi)) # Output: <class 'float'>
print(type(greeting)) # Output: <class 'str'>
print(type(my_list)) # Output: <class 'list'>
print(type(my_tuple)) # Output: <class 'tuple'>
print(type(my_dict)) # Output: <class 'dict'>
print(type(my_set)) # Output: <class 'set'>
print(my_set) # Output: {1, 2, 3}

1.3 Operators

Operators are special symbols used to perform operations on values (operands).

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division (float result)
  • % Modulus (remainder)
  • ** Exponentiation
  • // Floor division

Comparison Operators

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

Logical Operators

  • and Both conditions true
  • or At least one condition true
  • not Reverses the result

Other Operators

  • in Membership (sequence)
  • not in Not in sequence
  • is Identity (same object)
  • is not Not same object
# Arithmetic examples
a = 15
b = 4
print(f"{a} + {b} = {a + b}") # 19
print(f"{a} / {b} = {a / b}") # 3.75
print(f"{a} // {b} = {a // b}") # 3
print(f"{a} % {b} = {a % b}") # 3
print(f"{a} ** {b} = {a ** b}") # 50625

# Comparison examples
x = 10
y = 20
print(f"{x} == {y}: {x == y}") # False
print(f"{x} < {y}: {x < y}") # True

# Logical examples
age = 22
has_license = True
print(f"Can drive? {age >= 18 and has_license}") # True
print(f"Is minor or no license? {age < 18 or not has_license}") # False

Control Flow and Functions

2.1 Control Flow Statements

Conditional Statements (if, elif, else)

Allow your program to make decisions and execute different blocks of code based on conditions. Indentation is crucial in Python; it defines the code blocks.

temperature = 25

if temperature > 30:
    print("It's a hot day!")
    print("Stay hydrated.")
elif temperature > 20: # elif stands for "else if"
    print("It's a pleasant day.")
elif temperature < 10:
    print("It's quite cold.")
else:
    print("It's cool.")

# This line is outside the if/elif/else block
print("Weather check complete.")

Loops

Used to execute a block of code repeatedly.

for loop

Iterates over the items of any sequence (list, tuple, string, etc.)

# Looping through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"- {fruit}")
while loop

Executes as long as a condition remains True

count = 0
while count < 5:
    print(f"Count is {count}")
    count += 1 # Crucial step
print("Loop finished.")
Loop Control Statements
break

Immediately terminates the current loop

continue

Skips to the next iteration

for num in range(1, 11): # Numbers 1 to 10
    if num == 7:
        print("Breaking at 7")
        break # Exit the loop when num is 7
    if num % 2 == 0:
        continue # Skip even numbers
    print(f"Processing odd number: {num}")

2.2 Functions

Functions are named blocks of reusable code that perform a specific task. They help make code modular, organized, and easier to maintain (DRY - Don't Repeat Yourself).

Function Definition

# Defining a function with parameters and return value
def calculate_area(length, width):
    """Calculates the area of a rectangle."""
    if length < 0 or width < 0:
        return None # Indicate invalid input
    area = length * width
    return area

# Calling the function
rect_area = calculate_area(10, 5)
if rect_area is not None:
    print(f"The area is: {rect_area}")

Default Parameters

def power(base, exponent=2):
    """Calculates base raised to the power of exponent."""
    return base ** exponent

print(f"3 squared is: {power(3)}") # Uses default exponent 2 -> 9
print(f"2 cubed is: {power(2, 3)}") # Provides exponent 3 -> 8

Modules and OOP Concepts

3.1 Modules

Modules are Python files (.py extension) containing Python definitions that can be imported into other Python scripts.

Importing Modules

# Import entire module
import math
print(f"Square root of 16: {math.sqrt(16)}")

# Import specific items
from math import sqrt, pow
print(f"Square root of 25: {sqrt(25)}")

# Import with alias
import numpy as np
arr = np.array([1, 2, 3])

Creating Your Own Module

Create my_utils.py:

# my_utils.py
PI = 3.14159

def calculate_circumference(radius):
    """Calculates circle circumference."""
    return 2 * PI * radius

Then in main.py:

import my_utils
circum = my_utils.calculate_circumference(5)
print(f"Circumference: {circum}")

3.2 Object-Oriented Programming (OOP)

OOP structures code around "objects," which combine data (attributes) and behavior (methods).

Class Definition

# Define a class named 'Car'
class Car:
    # Class attribute (shared by all instances)
    wheels = 4

    # Constructor method
    def __init__(self, make, model, year, color):
        # Instance attributes
        self.make = make
        self.model = model
        self.year = year
        self.color = color
        self.is_running = False
        print(f"Created a {year} {make} {model}.")

    # Instance method
    def start_engine(self):
        if not self.is_running:
            self.is_running = True
            print(f"The {self.make} {self.model}'s engine is running.")

Creating and Using Objects

# Create instances (objects) of Car
my_car = Car("Toyota", "Camry", 2022, "Blue")
your_car = Car("Honda", "Civic", 2021, "Red")

# Access attributes
print(f"My car's make: {my_car.make}")
print(f"All cars have {Car.wheels} wheels.")

# Call methods
my_car.start_engine()
your_car.start_engine()

OOP Principles

  • Inheritance: Creating new classes from existing ones
  • Encapsulation: Bundling data with methods
  • Polymorphism: Same interface for different objects

Essential Libraries Overview

Python's power is significantly amplified by its vast ecosystem of third-party libraries. Here are some of the most commonly used ones:

NumPy

The fundamental package for scientific computing with Python. Provides support for large, multi-dimensional arrays and matrices.

Install: pip install numpy

Pandas

High-performance, easy-to-use data structures and data analysis tools. Essential for data cleaning and manipulation.

Install: pip install pandas

Matplotlib

A comprehensive library for creating static, animated, and interactive visualizations in Python.

Install: pip install matplotlib

Seaborn

A data visualization library based on Matplotlib that provides a high-level interface for statistical graphics.

Install: pip install seaborn

Requests

Makes HTTP requests simple and human-friendly. Used for interacting with web services (APIs).

Install: pip install requests

Flask & Django

Web development frameworks for building web applications. Flask is lightweight, Django is "batteries-included".

Install: pip install Flask / pip install Django

Scikit-learn

One of the most popular libraries for machine learning. Provides efficient tools for data analysis and ML tasks.

Install: pip install scikit-learn

How to Install Libraries

Use pip, the Python package installer, in your terminal or command prompt:

# Install multiple libraries at once
pip install numpy pandas matplotlib

# Install a specific library
pip install requests
